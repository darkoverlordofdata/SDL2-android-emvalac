# <del>ShmupWarz II</del> Better Than Shmup

vala-entitas running on android, emscripten and linux desktop.

dev dependencies: valac, npm, doran


## build (windows, linux)
    git clone https://github.com/darkoverlordofdata/zerog-shmupwarz.git
    cd zerog-shmupwarz
    npm install
    doran install
    ./configure
    mkdir build
    cd build
    make
